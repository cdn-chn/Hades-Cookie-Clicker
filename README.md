A full-stack, "Hades 2" inspired incremental clicker game. Players gather resources from the Underworld, purchase thematic upgrades, and compete for the highest score on a global leaderboard. 

## 🚀 Features
* **User Authentication:** Secure login and registration system using PHP sessions to track individual player progress and protect user data.
* **Core Gameplay Loop:** JavaScript-driven incremental mechanics where players click to earn resources and unlock permanent, thematic multipliers.
* **Global Leaderboard:** A dynamic scoreboard page that queries the backend to display the top player scores in real-time.
* **Persistent Data:** Player scores and upgrade states are continuously synced and saved to a backend MySQL database, allowing users to safely log out and resume their climb later.

## 💻 Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Backend:** PHP
* **Database:** MySQL (managed via phpMyAdmin)

## 🧠 What I Learned
Building this project bridged the gap between interactive front-end interfaces and back-end data persistence. It provided hands-on experience with managing user sessions in PHP, writing efficient SQL queries to sort/update leaderboards, and syncing dynamic JavaScript game states with a relational database.
